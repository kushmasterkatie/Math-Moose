Hello. This is my first post. Today I am going to explain to myself some of the elementary concepts related to topology. This includes what is in fact a topological space, metric spaces, compactness, and connectedness. In my next post I will write about some of the basics of algebra. This should cover all of my bases for an intro to Algebraic Topology.

Topological Spaces

A topological space is a set of points that have neighborhoods for each point, which satisfy a number of axoims. I will define the space using open sets. (This comes from Wikipedia)

A topological space is an ordered pair \((X,\mathfrak{T})\), where \(X\) is a set and \(\mathfrak{T}\) is a collection of subsets of X, satsifying the following axioms.

  1. The empty set \(\emptyset\) and \(X\) itself belong to \(\theta\).
  2. Any arbitrary union of members of \(\mathfrak{T}\) belongs to \(\mathfrak{T}\).
  3. The intersection of any finite number of members of \(\mathfrak{T}\) belongs to \(\mathfrak{T}\).

Connectedness

Connectedness is the notion that a topological space can't be the union of two or more disjoint (non-empty) spaces. In mathematical language:

A space is connected if there are no two open sets \(U\) and \(V\) such that \(U \cup V\) and \(U,V \neq \emptyset\).

Now the following conditions are equivalent for a topological space \(X\):

  • \(X\) is connected.
  • The only subsets of \(X\) which are both open and closed are \(X\) and \(\emptyset\). These kinds of sets are called clopen!
  • The only subsets of \(X\) with empty boundary are \(X\) and \(\emptyset\).
  • All continuous functions \(f: X \rightarrow \{0,1\}\) are constant and \(\{0,1\}\) is the two-point space endowed with the discrete topology.

Now there are some similar notions: path-connected and simply connected.

Def. A path-connected space \(X\) requires the existence of a path between any two points \(x,y \in X\), which is a continuous function \(f: [0,1] \rightarrow X\) such that \(f(0)=x\) amd \(f(1)=y\).
